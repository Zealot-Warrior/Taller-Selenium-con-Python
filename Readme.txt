¿Qué es Selenium?
Selenium es un conjunto de herramientas que automatiza navegadores web. Se utiliza principalmente para automatizar pruebas en aplicaciones web, simulando acciones como clics, introducción de texto y navegación entre páginas.

¿Quién soporta/mantiene Selenium?
Selenium es mantenido por una comunidad de desarrolladores y organizaciones, respaldado principalmente por Selenium Project, que recibe el apoyo de empresas como Google, Mozilla y otras que contribuyen al desarrollo de la herramienta.

¿Qué es WebDriver?
WebDriver es un componente de Selenium que permite controlar el navegador a través de una API, interactuando con el navegador a un nivel más profundo para realizar pruebas automatizadas.

¿Qué es ChromeDriver?
ChromeDriver es un ejecutable independiente que habilita el control del navegador Google Chrome a través de Selenium WebDriver, permitiendo la automatización de tareas en este navegador.

¿Qué licencia tiene Selenium?
Selenium está bajo la licencia Apache 2.0, lo que permite su uso libre y abierto para cualquier tipo de proyecto, incluso comerciales.

Para resolver una actividad de automatización, se utilizó ChromeDriver con una versión compatible del navegador Google Chrome que tenga el usuario. Esto asegura que la automatización funcione correctamente en el entorno local del usuario.
